# outputs/

Generated artifacts. Metrics CSVs and report figures are tracked in git; large model
and prediction artifacts are gitignored (reproducible from seeds, published to the
Hugging Face Hub at delivery).

## Metrics (tracked, canonical)

| File | Contents |
|---|---|
| `single_tier_prod.csv` | Single-tier, production target, Test A per-zone MAPE |
| `single_tier_cons.csv` | Single-tier, consumption target, Test A (the best model) |
| `e2_realsplit_testA_valsel.csv` | Two-tier production (CarbonCast-faithful), Test A, validation-selected |
| `e3_realsplit_testA_valsel.csv` | Two-tier consumption (CarbonCast-extended), Test A, validation-selected |
| `structural_comparison.csv` | Four-framework single vs two-tier comparison |
| `test_b_structural.csv` | Test B (live June 2026) per-config 1-24h scores |
| `test_b_headtohead.csv` | Test B head-to-head vs Electricity Maps (1-24h band) |
| `tier1_horizon_nrmse.csv` | Tier 1 total-generation forecast error (normalized RMSE) by horizon |

## Figures (tracked)

`figures/` holds the report figures (PDF), regenerated by `scripts/make_*.py` and
synced to Overleaf. The figures are tracked so the report compiles from a fresh clone.

## Artifacts (gitignored, local or Hugging Face Hub)

| Directory | Contents |
|---|---|
| `models/` | Two-tier consumption (E3) trained models per zone (reload with `load_e3`) |
| `models_single_prod/`, `models_single_cons/` | Single-tier trained models (`tier2.keras` + `meta.pkl`) |
| `preds/`, `preds_single_prod/`, `preds_single_cons/`, `preds_e2_prod/` | Saved predictions (`.npz`) per framework |

Note: two-tier production (E2) predictions live in `preds_e2_prod/`; its trained models
are not currently saved (regenerate via `carboncast_faithful` and `save` if needed).
